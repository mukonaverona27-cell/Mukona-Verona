It features integration with Google's Gemini API. **Live Demo:** https://mukonaverona27-cell.github.io/Mukona-Verona/ ## Features - ⚡ Fast development with Vite - 🎨 Styled with Tailwind CSS - 🤖 Powered by Google Gemini API - 📱 Responsive design - ✨ Modern React 19 ## Prerequisites - Node.js 18+ - npm or yarn ## Installation 1. Clone the repository: ```bash git clone https://github.com/mukonaverona27-cell/Mukona-Verona.git cd Mukona-Verona ``` 2. Install dependencies: ```bash npm install ``` 3. Create a `.env.local` file and add your Gemini API key: ```env GEMINI_API_KEY=your_api_key_here ``` ## Running Locally ```bash npm run dev ``` The app will start at http://localhost:3000 ## Building for Production ```bash npm run build ``` This generates optimized files in the `dist/` folder. ## Deploying to GitHub Pages The project is configured with GitHub Actions for automatic deployment to GitHub Pages. Every push to the `main` branch will automatically build and deploy the site to: **https://mukonaverona27-cell.github.io/Mukona-Verona/** ## Configuration Vite is configured with: - React plugin for JSX support - Tailwind CSS for styling - Base path set to `/Mukona-Verona/` for GitHub Pages ## License MIT
